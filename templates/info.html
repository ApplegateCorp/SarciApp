Informations pratiques

Tout ce qu'il faut savoir pour preparer votre sejour.

01

Lieu

Le festival se tient dans un cadre naturel prive — La Prairie. L'adresse exacte vous sera communiquee par email apres l'achat de votre billet.

02

Billets et acces

Le festival est sur invitation uniquement. Aucune vente de billet ne sera possible sur place. Votre QR code d'entree vous est envoye par email des l'achat et accessible a tout moment depuis votre compte.

Un seul scan par billet est autorise a l'entree.

{% if not user %} Acheter mon billet {% endif %}
03

Le bar et les consommations

Les bars du festival fonctionnent via un systeme de compte prepaye. Rechargez votre compte depuis votre telephone a tout moment, avant ou pendant le festival.

Au bar, le barman selectionne vos consommations et scanne votre QR code — le montant est deduit instantanement. Pas de liquide, pas de file d'attente.

{% if user %} Recharger mon compte {% endif %}
04

Comment utiliser l'application

  1. Creez votre compte et achetez votre billet en ligne.
  2. Recevez votre QR code par email (aussi accessible depuis votre compte).
  3. A l'entree : presentez votre QR code, il sera scanne une fois.
  4. Au bar : rechargez votre compte, presentez votre QR au barman.

Astuce : ajoutez ce site a votre ecran d'accueil pour un acces rapide (Safari — Sur l'ecran d'accueil).
