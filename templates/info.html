Billets et acces

Apres l'achat, creez votre compte sur la plateforme avec le meme email pour acceder a votre QR code et au systeme de bar prepaye.

03

Le bar et les consommations

Les bars du festival fonctionnent via un systeme de compte prepaye. Rechargez votre compte depuis votre telephone a tout moment, avant ou pendant le festival.

Au bar, le barman selectionne vos consommations et scanne votre QR code — le montant est deduit instantanement. Pas de liquide, pas de file d'attente.

{% if user %} Recharger mon compte {% endif %}
04

Comment utiliser l'application

  1. Achetez votre billet sur HelloAsso.
  2. Creez votre compte sur la plateforme avec le meme email.
  3. Votre QR code est genere automatiquement (accessible depuis votre compte).
  4. A l'entree : presentez votre QR code, il sera scanne une fois.
  5. Au bar : rechargez votre solde en ligne, presentez votre QR au barman.

Astuce : ajoutez ce site a votre ecran d'accueil pour un acces rapide (Safari — Sur l'ecran d'accueil).
